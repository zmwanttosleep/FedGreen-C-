### 3.1 自回归预测原理

**问题**：模型输出1个点 → 需要6个点

**解决方案**：用预测值作为输入继续预测
```
输入: [t-23, t-22, ..., t] → 输出: t+1
输入: [t-22, t-21, ..., t+1] → 输出: t+2
...
输入: [t-18, t-17, ..., t+5] → 输出: t+6
```

**误差累积问题**：每个步骤都会放大误差 → 需要**误差校正**

### 3.2 INT8量化原理

```
FP32:  0.12345678 (4字节) → 计算精确
INT8:  123 (1字节)      → 计算快速

量化公式：q = round(r / scale) + zero_point
压缩效果：模型大小 3.8MB → 1.2MB (压缩比 3.12x)
精度损失：仅 0.01%
```

### 5.1 特征工程模块

```python
class FeatureEngineer:
    """从单一功耗生成11个特征"""
    def transform(self, df):
        features = pd.DataFrame()
        features['power'] = df['power']                    # 原始
        features['hour'] = df['timestamp'].dt.hour / 24    # 时间
        features['day_of_week'] = df['timestamp'].dt.dayofweek / 6
        
        for window in [3, 6, 12]:                          # 滚动统计
            features[f'power_ma_{window}'] = df['power'].rolling(window).mean()
            features[f'power_std_{window}'] = df['power'].rolling(window).std()
        
        features['power_diff_1'] = df['power'].diff(1)     # 差分
        features['power_diff_2'] = df['power'].diff(2)
        return features
```

### 5.2 误差校正模块

```python
class ErrorCorrector:
    """基于历史误差动态校正预测值"""
    def __init__(self):
        self.correction_factors = np.zeros(6)
        self.error_history = []
    
    def update(self, predictions, actuals):
        errors = actuals - predictions
        self.error_history.extend(errors)
        if len(self.error_history) >= 100:
            # 计算每个时间步的平均误差
            errors_array = np.array(self.error_history[-100:]).reshape(-1, 6)
            self.correction_factors = np.mean(errors_array, axis=0)
    
    def correct(self, predictions, step):
        # 越靠后的时间步校正因子越大
        correction = self.correction_factors[step] * (1 + 0.1 * step)
        return predictions + correction
```

### 5.3 异常检测模块

```python
class AnomalyDetector:
    """实时检测功耗异常"""
    def __init__(self, threshold=3.0):
        self.threshold = threshold
        self.history = []
    
    def detect(self, value):
        if len(self.history) < 10:
            return False, 0.0
        
        mean = np.mean(self.history[-100:])
        std = np.std(self.history[-100:])
        z_score = abs(value - mean) / (std + 1e-8)
        
        is_anomaly = z_score > self.threshold
        self.history.append(value)
        
        return is_anomaly, z_score
```

### 5.4 性能监控模块

```python
class PerformanceMonitor:
    """追踪推理时间和误差趋势"""
    def __init__(self, window=100):
        self.times = []
        self.errors = []
        self.start = time.time()
    
    def log(self, time_ms, error):
        self.times.append(time_ms)
        self.errors.append(error)
        
    def get_stats(self):
        return {
            'avg_time': np.mean(self.times[-100:]),
            'p95_time': np.percentile(self.times, 95),
            'trend': self._calculate_trend(),
            'uptime': (time.time() - self.start) / 3600
        }
```
