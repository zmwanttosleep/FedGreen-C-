## 四、FedAvg 修复

### 问题
```python
AttributeError: 'FedClient' object has no attribute 'evaluate'
```

### 原因
`day4_fedavg_fixed.py` 中的 `FedClient` 类缺少 `evaluate` 方法，但在主训练循环中调用了它。

### 解决方案
添加 `evaluate` 方法：

```python
def evaluate(self, model):
    model.eval()
    criterion = nn.MSELoss()
    total_loss = 0.0
    count = 0
    
    with torch.no_grad():
        for X_batch, y_batch in self.test_loader:
            X_batch, y_batch = X_batch.to(device), y_batch.to(device)
            output = model(X_batch)
            loss = criterion(output, y_batch)
            total_loss += loss.item()
            count += 1
    
    return total_loss / count, {'test_loss': total_loss / count}
```

### 验证
✅ FedAvg 已可正常运行
