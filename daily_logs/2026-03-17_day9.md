## 三、核心原理：FedProx

**生动类比：高考移民监管**

| 角色 | FedAvg（无监管） | FedProx（有监管） |
|:---|:---|:---|
| 本地学校 | 自由调整教案 | 必须在教育局大纲范围内调整 |
| 统一考卷 | 无 | 全局模型作为统一标准 |
| 违规成本 | 无 | 偏离大纲越远，惩罚越大 |

**专业解释**

FedProx在FedAvg的基础上增加近端项（proximal term），防止本地模型偏离全局模型太远：

```
Loss_local = MSE(y_pred, y_true) + (μ/2) * ||w - w_global||²
            ↑ 原有预测任务        ↑ 惩罚项
```

**μ参数的作用**：
- μ=0：退化为FedAvg（完全自由）
- μ=0.001：轻微约束（仅防止大幅偏离）
- μ=0.01：中等约束（平衡全局与局部）
- μ=0.1：强约束（趋近全局模型）

## 八、核心代码

### 数据加载器（按邮编区分节点）
```python
class BarcelonaEnergyLoader:
    def __init__(self, data_dir, seq_len=24, pred_len=1):
        self.data_dir = data_dir
        self.seq_len = seq_len
        self.pred_len = pred_len
    
    def get_node_loaders(self, node_ids, batch_size=64):
        """返回每个节点的train/val/test DataLoader"""
        loaders = {}
        for node in node_ids:
            # 加载该邮编区数据
            node_data = self._load_node_data(node)
            
            # 按时间顺序划分70/10/20
            train, val, test = self._split_by_time(node_data)
            
            loaders[node] = {
                'train': self._create_loader(train, batch_size),
                'val': self._create_loader(val, batch_size),
                'test': self._create_loader(test, batch_size)
            }
        return loaders
    
    def _create_loader(self, data, batch_size):
        """生成滑动窗口序列 (24小时→1小时)"""
        X, y = [], []
        for i in range(len(data) - self.seq_len - self.pred_len + 1):
            X.append(data[i:i+self.seq_len])
            y.append(data[i+self.seq_len:i+self.seq_len+self.pred_len])
        dataset = TensorDataset(torch.FloatTensor(X), torch.FloatTensor(y))
        return DataLoader(dataset, batch_size=batch_size, shuffle=True)
```

### FedProx本地训练
```python
def train_local(model, global_model, train_loader, mu, epochs=3, lr=0.01):
    """带近端项的本地训练"""
    optimizer = torch.optim.SGD(model.parameters(), lr=lr)
    global_params = [p.detach().clone() for p in global_model.parameters()]
    
    for epoch in range(epochs):
        for x, y in train_loader:
            optimizer.zero_grad()
            
            # 前向传播
            pred = model(x)
            mse_loss = F.mse_loss(pred, y)
            
            # 计算近端项
            proximal_term = 0
            for w, w_global in zip(model.parameters(), global_params):
                proximal_term += (mu / 2) * torch.norm(w - w_global)**2
            
            # 总损失
            loss = mse_loss + proximal_term
            loss.backward()
            optimizer.step()
    
    return model.state_dict()
```
