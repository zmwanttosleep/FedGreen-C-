## 三、核心原理（类比+专业）

### 3.1 LSTM的门控机制

**生动类比：从普通员工到经理的记事本**

| 特性 | RNN（普通员工） | LSTM（经理） |
|:---|:---|:---|
| 短期记忆 | 只能记住最近几句话 | 有专门的工作笔记 |
| 长期记忆 | 早会内容下午就忘 | 重要决策写入档案 |
| 信息筛选 | 来者不拒，信息过载 | 门卫过滤，只记关键 |

**专业解释**
LSTM通过三个门控制信息流：
- **遗忘门**：决定丢弃哪些旧信息
- **输入门**：决定存储哪些新信息
- **输出门**：决定输出哪些信息

**记忆口诀**：先忘后记，再输出

---

### 3.2 单变量 → 多变量

**生动类比：从单一侦探到专案组**

| 场景 | 单变量 | 多变量 |
|:---|:---|:---|
| 侦探人数 | 1人（只看能耗） | 3人专案组（能耗+温度+流量） |
| 信息来源 | 单一监控 | 多维度情报网 |
| 判断依据 | "能耗突然高了" | "能耗高+温度正常+流量低=设备故障" |
| 预测准确度 | 受限于信息茧房 | 交叉验证，更准确 |

### 3.4 FedAvg联邦学习

**生动类比：五个学校的期末考试**

| 角色 | 类比 | 作用 |
|:---|:---|:---|
| 各站点 | 各个学校 | 各自出题、各自阅卷 |
| 本地模型 | 学校自己的题库 | 针对本校学生优化 |
| 全局模型 | 教育局统一教材 | 聚合各校经验 |
| FedAvg | 教研组会议 | 平均各校的优秀教案 |

**聚合公式**：
```
w_global = Σ (n_i / n_total) × w_i
```
- n_i: 第i个站点的样本数
- w_i: 第i个站点的模型参数

---

### 3.5 GCN图卷积网络

**生动类比：小区业主委员会**

| 角色 | 类比 | 作用 |
|:---|:---|:---|
| 节点 | 每户业主 | 有自己的特征（面积、人口） |
| 边 | 邻里关系 | 相互交流信息 |
| 邻居聚合 | 楼长收集意见 | 汇总邻居信息 |
| 多层GCN | 业委会层层传达 | 信息在社区传播 |

**核心公式**：
```
H^{(l+1)} = σ(D^{-1/2} A D^{-1/2} H^{(l)} W^{(l)})
```

---

### 3.6 GAT图注意力网络

**生动类比：班级评选三好学生**

| 角色 | 类比 | 作用 |
|:---|:---|:---|
| 节点 | 每个学生 | 有自己的成绩、品德 |
| 邻居 | 同班同学 | 互相评价 |
| 注意力权重 | 投票权重 | 关系好的同学权重高 |
| 多头注意力 | 多个评委 | 综合多方意见 |

**注意力机制**：
```
α_{ij} = softmax(LeakyReLU(a^T [W h_i || W h_j]))
```

## 八、代码存档

### 8.1 LSTM模型定义
```python
class LSTMPredictor(nn.Module):
    def __init__(self, input_dim=3, hidden_dim=64, num_layers=2):
        super().__init__()
        self.lstm = nn.LSTM(input_dim, hidden_dim, num_layers, batch_first=True)
        self.fc = nn.Linear(hidden_dim, 1)
    
    def forward(self, x):
        lstm_out, _ = self.lstm(x)
        last_hidden = lstm_out[:, -1, :]
        return self.fc(last_hidden)
```

### 8.2 FedAvg核心代码
```python
def fedavg_aggregate(local_models, weights):
    """联邦平均聚合"""
    global_model = copy.deepcopy(local_models[0])
    global_dict = global_model.state_dict()
    
    for key in global_dict.keys():
        global_dict[key] = sum(w * m.state_dict()[key] 
                              for w, m in zip(weights, local_models))
    
    global_model.load_state_dict(global_dict)
    return global_model
```

### 8.3 GCN模型定义
```python
class GCN(torch.nn.Module):
    def __init__(self, hidden_channels=16):
        super().__init__()
        self.conv1 = GCNConv(dataset.num_features, hidden_channels)
        self.conv2 = GCNConv(hidden_channels, dataset.num_classes)
    
    def forward(self, x, edge_index):
        x = self.conv1(x, edge_index)
        x = F.relu(x)
        x = F.dropout(x, p=0.5, training=self.training)
        x = self.conv2(x, edge_index)
        return x
```

### 8.4 GAT模型定义
```python
class GAT(torch.nn.Module):
    def __init__(self, in_channels, hidden_channels, out_channels, heads=8, dropout=0.6):
        super().__init__()
        self.conv1 = GATConv(in_channels, hidden_channels, heads=heads, dropout=dropout)
        self.conv2 = GATConv(hidden_channels * heads, out_channels, heads=1, concat=False)
        self.dropout = dropout
    
    def forward(self, x, edge_index):
        x = self.conv1(x, edge_index)
        x = F.elu(x)
        x = F.dropout(x, p=self.dropout, training=self.training)
        x = self.conv2(x, edge_index)
        return x
```
